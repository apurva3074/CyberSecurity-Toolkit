# Extension Icons

The extension requires three icon sizes:

- icon16.png (16x16) - Shown in the toolbar
- icon48.png (48x48) - Shown in extension management
- icon128.png (128x128) - Shown in Chrome Web Store

## Quick Icon Creation:

You can create simple icons using:

1. **Canva** (free, easy)
2. **Figma** (more professional)
3. **Online Icon Generators**: https://favicon.io/

## Temporary Solution:

For now, you can use a simple colored square or your logo. The extension will work without icons, but Chrome will show a default puzzle piece icon.

## Suggested Design:

- Purple/gradient background (matching your theme)
- Shield or lock icon (security theme)
- Letter "P" or "URL" text
